## 技術スタック

- フロントエンド
  - Tanstack Start
  - Tailwind + shadcn
  - TypeScript
- バックエンド
  - Cloud Functions
  - TypeScript
- インフラ
  - Firebase App Hosting
- データベース
  - Firestore
- オブジェクトストレージ
 - Cloud Storage for Firebase
- 認証認可
  - Firebase Auth
