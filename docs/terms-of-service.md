# LGTMonk 利用規約

最終更新日: 2026年3月11日

## 第1条（適用）

1. 本規約は、LGTMonk（以下「本サービス」といいます）の利用に関する条件を定めるものです。
2. 本サービスを利用するすべてのユーザーは、本規約に同意したものとみなされます。

## 第2条（サービスの内容）

本サービスは、GitHub のプルリクエストに使用する LGTM 画像を生成・検索・コピーできる Web アプリケーションおよび Chrome 拡張機能を提供します。

主な機能：
- LGTM 画像の一覧表示（最新順・ランダム・お気に入り）
- 画像のクリップボードへのコピー
- 画像アップロードによる LGTM 画像生成
- Google 画像検索を利用した LGTM 画像生成
- キーワードによる画像検索
- お気に入り機能（ログイン時は永続保存）

## 第3条（利用資格）

1. 本サービスは、GitHub でコードレビューを行うエンジニアを主な対象としています。
2. 未ログインでも基本機能は利用できますが、一部機能（生成履歴の永続保存、お気に入りの永続保存）はログインが必要です。
3. ユーザー認証には Firebase Authentication（Google アカウント）を使用します。

## 第4条（禁止事項）

ユーザーは、本サービスの利用にあたり、以下の行為を行ってはなりません：

1. 法令または公序良俗に違反する行為
2. 犯罪行為に関連する行為
3. 他者の著作権、商標権、肖像権などの知的財産権を侵害する行為
4. 第三者の名誉、信用、プライバシーを侵害する行為
5. わいせつ、暴力的、差別的、その他不適切な内容を含む画像のアップロード
6. 本サービスのサーバーやネットワークに過度な負荷をかける行為
7. 本サービスの運営を妨害する行為
8. 不正アクセス行為、またはこれを試みる行為
9. 本サービスの脆弱性を利用または悪用する行為
10. 本サービスを商用目的で利用する行為（個人的な GitHub での利用は除く）
11. その他、当サービスが不適切と判断する行為

## 第5条（画像の利用について）

1. **アップロード画像**: ユーザーがアップロードする画像は、ユーザー自身が権利を有するもの、または利用許諾を得たものに限ります。
2. **Google 画像検索**: Google 画像検索で取得した画像を使用する場合、著作権等の権利関係はユーザーの責任で確認してください。
3. **生成画像の利用**: 生成された LGTM 画像は、GitHub のプルリクエストでの利用を想定しています。
4. **公開範囲**: 生成された画像は公開 URL として保存され、URL を知る誰もがアクセス可能です。機密情報を含む画像はアップロードしないでください。

## 第6条（著作権）

1. 本サービスの著作権は、開発者に帰属します。
2. ユーザーがアップロードした画像の著作権は、元の権利者に帰属します。
3. 生成された LGTM 画像（「LGTM」テキストが合成された画像）は、元画像の著作権と本サービスの加工による著作権が存在します。

## 第7条（データの保存と削除）

1. 生成された画像は Cloud Storage for Firebase に保存されます。
2. 画像のメタデータ（URL、キーワード、コピー回数など）は Firestore に保存されます。
3. 現時点では、ユーザーによる画像削除機能は提供していません。
4. 運営者は、以下に該当すると判断した画像を、ユーザーへの事前通知なく削除できるものとします：
   - 第4条（禁止事項）に違反する画像
   - 法令、公序良俗に反する画像
   - 著作権、肖像権等の権利を侵害する画像
   - わいせつ、暴力的、差別的、その他不適切な内容を含む画像
   - その他、運営者が不適切と判断した画像
5. 前項に基づく画像削除により生じた損害について、運営者は一切の責任を負いません。

## 補足：Chrome 拡張機能について

Chrome 拡張機能版も、本規約が適用されます。

拡張機能の特有事項：
- GitHub のプルリクエスト画面からワンクリックで LGTM 画像を取得・貼り付けできます
- 拡張機能は LGTMonk API を呼び出してランダムな画像を取得します
- 画像のコピー回数は Firestore に記録されます（匿名）
- プライバシーポリシーは別途ご確認ください
